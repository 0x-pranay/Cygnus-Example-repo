 Using Git

1. Git install in our machine
2. [one time ] Configure email and name  -->   to include in the commit.
3.  Initialise git  ->  Asking git to keep track of my directory
4. make some changes
5. move them to staging
6. commit them 
7. using branches to seperatly work on different features.


# Branches

1. create a new Branch
	git branch <branch_name>
		or
  if you  want to create a branch and checkout to that branch use
	git checkout -b <branch_name>

2. changes -> add to staging -> commit
3. If you want this branch to merge with another branch use merging.
